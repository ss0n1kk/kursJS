Обучающий сайт-игра по JS. Стек: HTML/CSS/JS + Bootstrap5

index.html - главная страница
prologue.html - введение сюжет игры
map.html - страница связывающая все модули обучения, "карта" обучения
FirstQuest/prologue.html - развитие сюжета и переход к изучению теории
FirstQuest/grimuar.html - теория по JavaScript + мини-тесты на закрепление материала
FirstQuest/test.html - итоговая проверка знаний в виде Тестов от магистра в количестве шесть штук
FirstQuest/tavern.html - следующая локация, общение с игровым персонажем, взятие первого квеста
FirstQuest/TEST_code_final.html - выполнение квеста, практика написания кода
FirstQuest/reward.html - общение с игровым персонажем в таверне получение награды за выполненный квест
FirstQuest/final-dream.html - завершение модуля, развития сюжета, логический конец цепочки модуля, форма обратной связи
