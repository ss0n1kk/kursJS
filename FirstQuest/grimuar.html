Руна let — Сосуд изменчивой магии

let создаёт сосуд, содержимое которого можно менять. Как кошель для золота или флакон для зелья.

Java Script
let имяСосуда = значение;
Java Script
let health = 100; // Твои текущие очки жизни
let gold = 50; // Количество золота в кошельке

health = 75; // После схватки с гоблином жизнь уменьшилась
gold = 80; // Нашел клад — золота стало больше!

📜 Название сосудов важно писать по определенным правилам:

  • ✓ Только Англ. Буквы и цифры (и символы $ _ )
  • ✗ Начинать название с цифры вызовет ОШИБКУ
  • ✗ Пробелы запрещены, но можно использовать разный регистр букв
  • ⚠️ Регистр ОЧЕНЬ важен, сосуды “gold” и “Gold” - это два разных сосуда для системы

Выберите ТОЛЬКО ВЕРНЫЕ названия сосудов

  • 1. Золото
  • 2. 1золото
  • 3. gold
  • 4. 1gold
  • 5. Poison1
  • 6. HealPoison
  • 7. Heal Poison
  • 8. HOUSEcarpet
ответ запишите числами в порядке увеличения (пример: 1234)

Руна const — Сосуд вечной магии

const создаёт сосуд, привязанный к одному значению навсегда. Как твоё истинное имя или магическая печать.

Java Script
const имяСосуда = значение;
Java Script
const myTrueName = "Артас"; // Истинное имя изменить нельзя
const birthYear = 1223; // Год рождения — константа
const isMagician = true; // Ты маг, и это факт

Важно: Попытка изменить const вызовет магический сбой (ОШИБКУ)!

Комментарии в коде

Кроме кода, в файлах с исходным кодом могут находиться комментарии. Это текст, который не является частью программы и нужен программистам для пометок. С их помощью добавляют пояснения, как работает код, какие здесь ошибки нужно поправить или не забыть что-то добавить позже.

Java Script | Синтаксис заклинания
const имяСосуда = значение;
Java Script | Пример
const myTrueName = "Артас"; // Истинное имя изменить нельзя
const birthYear = 1223;
const isMagician = true;

Java Script | Комментарии в коде (расширенный гримуар)

Комментарии помогают понять магические формулы и предостерегают от неверных действий. Они бывают однострочные и многострочные.

🪞 Магическое зеркало (консоль)

Консоль помогает увидеть результат действия в Java Script и вывести его в консоль вашего сайта, можно использовать как проверку работоспособности функции(познакомимся позже) или выводить значения ваших сосудов на ваш сайт. Например, у вас есть сосуд PoisonBottle со значением 3, написал console.log(PoisonBottle) вы увидите в консоли цифру 3. Открыть консоль -> F12 -> Консоль

Java Script | Синтаксис призыва зеркала:
console.log(значение) // значение может быть любое, но текст(например русские буквы или сообщение) нужно заключать в ``
Java Script | Пример вызова зеркала:
let PoisonBottle = 3;
console.log(PoisonBottle); // пример вывода значения сосуда

console.log(`Привет, Мир!`); // пример вывода текста, обратите внимание на синтаксис
Что выведется в консоль?
console.log(`Привет, Мир!`); // пример вывода текста, обратите внимание на синтаксис

📜 Рунический Свод Правил:

  • Имя сосуда (идентификатор) может содержать буквы, цифры, символы $ и _.
  • Первый символ не может быть цифрой.
  • Регистр букв важен: Gold и gold — два разных сосуда.
  • Используй разницу регистра вместо пробелов для имён из нескольких слов, например: magicCrystalCount.
